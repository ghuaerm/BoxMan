# 基于C++的推箱子游戏
## 推箱子和地图编辑器


[/output/BoxMan.exe][game]：编译好的程序，Windows下可直接运行

[/output/BoxManEditer.exe][editor]：地图编辑器，可以对地图进行修改或添加
